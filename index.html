Beginners Tutorials

Start A New Frame

  1. Enter the left span, right span, and column height.
  2. Click Generate / Update Frame. The app always creates exactly two beams and three fixed-base columns.
  3. Enter beam and column section dimensions, then click Apply Sections.
  4. Use Select Member before assigning UDL or member point loads.
  5. Use Select Node before assigning node forces or moments.
  6. Click Calculate to update reactions, member results, and diagrams.
  7. Use Show/Hide Loading to simplify the drawing view.
  8. Use Print PDF or Print Local for report output.

Load Notes

DL, SDL, LL, WL, and EQL can be assigned as load cases. Beam and column self weight stays active in computation even when self-weight graphics are hidden.

Checking Notes

Always review supports, load direction, member sizes, and reaction balance before using results for design documentation.

Important

This tool is for modeling assistance. Final structural calculations must be checked and approved by a licensed Structural or Civil Engineer.
