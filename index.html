Beginners Tutorials

Start A New Frame

  1. Click New Design to clear the drawing area.
  2. Enter vertical grid positions in meters, such as 0,3,6,9.
  3. Enter horizontal levels in meters, such as 0,1.5,4.5.
  4. Click Generate Grid.
  5. Use Node Mode and click grid intersections to create nodes.
  6. Use Create Member Mode and click two nodes to create a beam or column.
  7. Use Support Mode, choose the support type, then click base nodes.
  8. Select members before assigning UDL or point loads.
  9. Select a node before assigning node P loads or nodal moments.
  10. Click Calculate to update reactions, member results, and diagrams.
  11. Use Show/Hide Loading to simplify the drawing view.
  12. Use Print PDF or Print Local for report output.

Load Notes

DL, SDL, LL, WL, and EQL can be assigned as load cases. Beam and column self weight stays active in computation even when self-weight graphics are hidden.

Checking Notes

Always review supports, load direction, member sizes, and reaction balance before using results for design documentation.

Important

This tool is for modeling assistance. Final structural calculations must be checked and approved by a licensed Structural or Civil Engineer.
