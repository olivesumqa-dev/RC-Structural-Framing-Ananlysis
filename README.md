# StrucForge 2D Frame Analysis

Static browser app for drawing 2D structural frame layouts, assigning supports and loads, and previewing shear and moment diagrams.

## Run Locally

Open `index.html` directly in a modern browser, or serve this folder with any static file server.

## Main Features

- Grid-based node and member creation
- Column grid bubbles and level/dimension annotations
- Top-right theme icon buttons, text-size control, and drawing legend
- Member labels with automatic section/material properties, node loads, UDLs, self-weight, and point loads drawn on the frame diagram
- Local browser records plus JSON Save, Save As, Export, and Load File
- Calculate button with toggleable one-window shear/moment diagrams for all frame levels and bays
